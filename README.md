Zeek Labs

A hands-on cybersecurity portfolio demonstrating network traffic analysis and digital forensics using Zeek, Wireshark, Docker, and Kali Linux.

About

This repository contains practical investigations performed on packet captures (PCAPs). Each lab demonstrates how Zeek converts raw network traffic into structured logs that support threat hunting, incident response, and forensic analysis.

The goal of this project is to build practical experience analyzing network activity and documenting investigations in a professional format.
